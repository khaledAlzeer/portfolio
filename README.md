# Khaled Alzeer — Portfolio 🚀

> AI Student & Developer · Al-Zaytoonah University of Jordan

A personal portfolio website showcasing my projects, skills, and certificates.
Built from scratch with pure HTML, CSS, and JavaScript — no frameworks.

---

## ✨ Features

- 🎨 **Glassmorphism Design** — dark theme with Blue · Cyan · Violet accents
- 🖱️ **Custom Cursor** — dot + ring with smooth lerp animation
- ⌨️ **Typewriter Effect** — animated role titles in the hero section
- 📜 **Scroll Reveal** — elements fade up as you scroll
- 📱 **Fully Responsive** — mobile drawer navigation
- ♾️ **Skills Marquee** — infinite scrolling technology ticker
- 🗂️ **Skills Tabs** — Languages · Frameworks · Tools
- 🌙 **Ambient Background** — animated floating blobs with dot grid
